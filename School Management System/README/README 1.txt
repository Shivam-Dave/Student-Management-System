School Management System (PHP & MySQL)
======================================

Overview :
----------
The School Management System is a PHP-based application designed to streamline school operations and enhance administrative efficiency. It seamlessly integrates with a locally hosted database, providing robust data management and security.


Features :
----------
Admin Login: Secure authentication ensures authorized access to the system.
Dynamic Editing: Easily manage and update class information, subjects, staff details, exam information, and an editable exam timetable.
Local Database Connectivity: Connects to a locally hosted database for efficient data handling, reducing latency and ensuring reliability.


Getting Started :
-----------------
>Clone the repository: git clone https://github.com/Shivam-Dave/School-Management-System.git

>Set up a local server environment (e.g., XAMPP, WampServer).

>Import the provided SQL script into your database to create the necessary tables.

>Configure database connection parameters in the PHP files.

>Open the application in your web browser.


Usage :
--------
Navigate to the admin login page and enter your credentials.(username : dave , password : 1234)
Explore dynamic editing capabilities for managing various aspects of the school, such as class information, subjects, staff details, exam information, and exam timetables.
